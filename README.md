# Fruit Sniffers — FSFR Token

## Overview
Fruit Sniffers (FSFR) is a fun and collectible ERC‑20 token designed as the glue of the **Fruit Sniffers Universe** — stickers, plushies, NFTs, merch, and scented art toys.

## Token Specs
- **Name:** Fruit Sniffers  
- **Symbol:** FSFR  
- **Decimals:** 18  
- **Total Supply:** 100,000,000 (fixed)  
- **Minting:** One‑time mint to deployer  
- **Admin:** None (no owner, no privileged functions)  
- **Transferability:** Fully standard ERC‑20  

## Lore
FSFR is a playful token born from the world of scratch‑and‑sniff fruits.  
Each token echoes the spirit of the fruity mascots — cute, mischievous, rebellious, sweet, sour, and always collectible.

## Usage
- Collectible branding token for Fruit Sniffers  
- NFT universe + plush toy brand anchor  
- Unlockable perks in future mini‑apps  
- Community rewards
